2우리의 진짜 무기 = 치료 사례 데이터

3고가 진료의 숫자 감각 — CPA·ROAS

고가 광고의 흔한 실수는 "클릭 단가가 비싸다"고 겁먹는 것. 핵심은 "상담 1건(리드)에 얼마까지 써도 남는가"다.

우리 숫자로 본 "리드 1건 허용 비용(목표 CPA)"
  • 치료 1건 매출 = 1,000만원
  • 상담 → 치료 전환율(보수적 가정) = 20% → 치료 1건당 상담 5건 필요
  • 마케팅 허용 비용(매출의 10~20%) = 100만 ~ 200만원/치료
  • → 상담(리드) 1건 허용 비용 = 100~200만원 ÷ 5 = 약 20만 ~ 40만원
  • 즉 상담 리드당 20~40만원까지 써도 ROAS 5~10배. "리드당 5만원"이면 엄청 싼 것!

저가 상품은 리드당 몇 천원이어야 하지만, 고가 진료는 리드당 수십만원도 흑자다. "클릭 단가"가 아니라 상담 수·CPA·상담의 질을 보라. 단, 광고가 데려온 상담이 실제 치료로 이어지는지(상담→치료 전환율)를 같이 추적해야 한다.

4오가닉 × 유료 섞는 법

  1. 오가닉으로 먼저 올린다 → 반응 좋은 게시물만 골라 부스팅(검증된 소재에만 돈). = 채널 피드 부스팅
  2. 콜드 도달은 다크포스트 영상으로 (오가닉으론 안 닿는 신규).
  3. 영상 본 사람·방문자·계산기 사용자가 쌓이면 → 그들에게만 리타게팅으로 상담 유도. 전환은 데워진 사람에게.
예산 원칙 (고가 기준) 작게 테스트로 소재·타겟 검증 → 학습단계 인내(첫 며칠 건드리지 말기) → 검증된 것 70% / 테스트 20% / 신규 10% → 잘돼도 한 번에 2배 금지(20~30%씩 증액) → 빈도 2~4 허용(반복이 신뢰), 5+면 소재 교체 → 리타게팅 풀(영상 시청자·방문자) 적립이 미래 전환의 연료.

5홈페이지 = 전환 허브 + LINE

치료사례: 버튼 4개 전환

스와이프 배너는 안 읽힌다 → 버튼 4개로 고정(능동 클릭·체류↑). 4개는 고민 유형이 다르게(저신장 / 성조숙증 / 비만동반 / 시기놓칠뻔) → 부모가 "우리 애 케이스"를 찾게.

버튼 라벨 = 결과 미끼: "OO 사례" 보다 "키 백분위 15% → 60% 따라잡은 케이스"(규정 내 톤).

그 아래 LINE 게이트 리드

4개는 무료 맛보기, 그 아래가 게이트:

"우리 아이 또래·같은 고민 사례를 더 보고 싶으세요? → LINE으로 전체 사례집 받아보기"

익명 방문자 → LINE 리드로 바꾸는 다리. 계산기 결과 화면에도 같은 CTA 한 번 더.

예측키 계산기 릴스 → 광고 (1순위 추천)

계산기는 무료·저저항 미끼다. "치료받으세요"(거부감↑)가 아니라 "우리 아이 18세 예측키 1분 무료 측정"(호기심↑) → 차가운 태국 타겟에 딱. 측정한 사람 = 의도 있는 리드 → 리타게팅 풀.

  • 광고 목표: 트래픽(계산기로) 또는 영상조회(콜드 인지)
  • 흐름: 릴스("걱정되시나요? 1분 측정") → 홈페이지 계산기 → 결과 → LINE 상담 CTA
  • 광고의 임무 = "측정하게" 만드는 것(낮은 약속). 전환은 그 뒤 퍼널이 한다.
  • 이미 Remotion에 계산기 릴 자산 있음(HeightReelsTH 등) → 새로 안 만들고 태국 버전 다듬어 사용.

LINE = 리드/육성 허브 (태국, @894qhqtu)

친구추가 → 자동 환영 + 사례집/계산기 안내 → 주기적 broadcast(새 사례·성장팁) → 1:1 상담 → 예약 → 치료.
