이 가이드는 인스타 피드에 "더 알아보기" 버튼이 달려 홈페이지로 바로 가는 광고를 Meta 광고 관리자에서 직접 만드는 법입니다. 이건 앱 검수와 무관하게 지금 됩니다(검수는 우리 dflo 앱에서 자동화할 때만 필요). 접속: business.facebook.com/adsmanager (master 계정 로그인).
사전 준비 1. 캠페인(목표) 2. 광고세트(타겟·예산) 3. 광고(버튼·링크) 4. 검토·게시 운영 순서 규정

사전 준비 (한 번만)

  • Meta 광고 관리자 접속 + 광고 계정 + 결제수단 등록
  • 페이스북 페이지 ↔ 인스타그램(프로페셔널 계정) 연결 (이미 연결됨 — 한국/태국)
  • Meta 픽셀 999009859491958 설치됨 — 홈페이지에서 PageView · Lead(메신저 클릭) · HeightCalcComplete(예측키 측정) 이벤트 발사 중. 이벤트 관리자에서 들어오는지 한 번 확인.
  • 랜딩 페이지 정하기: 보통 홈페이지 또는 예측키 계산기 (예: dr187growup.com · 태국 /th · 베트남 /vi)

STEP 1 — 캠페인: 목표 고르기

1

+ 만들기 → 캠페인 목표 선택

고가 진료라 2단계로 나눠 씁니다. 차가운 사람에게 바로 전환을 노리지 않습니다.

단계목표 선택성과 목표(최적화)언제
데우기트래픽(Traffic)랜딩 페이지 조회 수 최대화처음 — 콜드 유입 + 픽셀 모수 적립
전환잠재 고객(Leads)전환수 최대화 → 이벤트 Lead데워진 사람(리타게팅)에게

특별 광고 카테고리: 해당 없음(None) 선택. (성장클리닉은 신용·고용·주택·사회이슈가 아님 — 단, 건강 광고 정책은 STEP 3·규정 참고.)

STEP 2 — 광고 세트: 전환 위치 · 타겟 · 노출 위치 · 예산

2

전환 위치 (전환 캠페인일 때)

웹사이트 선택 → 픽셀 999009859491958 + 전환 이벤트 Lead. (트래픽 캠페인이면 이 단계 없이 다음으로.)

예산 및 일정

  • 일 예산으로 소액부터 (테스트: 태국 ฿300~500 / 베트남 ₫15~30만 / 미국 $10~20 정도)
  • 시작일 지정. 첫 3~7일은 학습 단계 — 예산·타겟 건드리지 말기(건드리면 학습 리셋).

잠재 고객(타게팅) — 우리 앱 🎯 타겟에서 정한 값을 그대로 입력

  • 위치: 도시·반경 (예: Bangkok +20km / 미국은 한인 밀집지 핀포인트)
  • 연령: 부모 30~45 · 성별: 여성 중심
  • 상세 타게팅: 관심사로 소득 프록시(국제·사립학교, 프리미엄 브랜드, 해외여행) + 육아·자녀교육·소아건강. ※ 특정 질환 등 민감 카테고리는 Meta가 없앴으니 일반 관심사로.
  • 맞춤 타겟(리타게팅): 전환 캠페인은 여기서 웹사이트 방문자·영상 시청자·계산기 완료자를 포함 / 기존 고객은 제외. 유사타겟(1~3%)도 여기.
  • Advantage+ 잠재고객: 처음엔 끄거나 "제안"으로 두고 직접 타겟을 통제 → 데이터 쌓이면 켜기.

노출 위치 (Placements)

수동 노출 위치 → Instagram의 피드 · 릴스 · 스토리 선택(인스타 집중). 넓게 가려면 Advantage+ 노출 위치(자동).

STEP 3 — 광고: 크리에이티브 + "더 알아보기" 버튼 + 링크

3

형식 & 미디어

단일 이미지/동영상 또는 캐러셀. 우리 카드뉴스·릴스를 업로드하거나 기존 인스타 게시물 사용. (※ 기존 게시물을 그대로 쓰면 캡션 링크가 클릭 안 되니, 아래 "행동 유도 + URL"을 꼭 채워 광고용 링크를 답니다.)

★ 여기가 핵심 — 직접 홈피로 보내는 부분

  • 기본 문구(캡션) · 제목 입력 (예: "우리 아이 18세 예측키, 1분 무료 측정")
  • 행동 유도(Call to Action): "더 알아보기" 선택
  • 웹사이트 URL: 랜딩(홈페이지 / 예측키 계산기) — 이게 있으면 피드 광고에 클릭 버튼이 생겨 바로 직행합니다.
이게 "프로필 bio 링크 경유"를 없애는 부분 링크 없는 일반 게시물은 인스타에서 캡션 링크가 안 눌려 → 프로필 가서 bio 링크로만 들어올 수 있어 대부분 이탈합니다. 광고에서 행동 유도 버튼 + URL을 채우면 피드에서 버튼 한 번에 홈피 직행이 됩니다.

추적

  • 픽셀 켜기 (999009859491958) — 방문·전환 측정.
  • URL에 UTM 붙이기 권장: ?utm_source=instagram&utm_medium=paid&utm_campaign=... → GA4·사이트 분석에서 광고 유입 구분.

STEP 4 — 검토 · 게시

4

미리보기로 인스타 피드 모양 확인 → 게시. Meta 심사(보통 수십 분~하루) 후 노출 시작. 게시 후 첫 며칠은 학습 단계 — 손대지 말고 CTR·CPC·랜딩 조회·메신저 클릭(Lead)을 지켜봅니다.

운영 순서 (고가 퍼널)

  1. 트래픽 캠페인으로 소액 콜드 테스트 → 이긴 소재 선별 + 리타게팅 풀(방문자·영상 시청자·계산기 완료자) 적립
  2. 예산 20~30%씩 증액(2배 금지)
  3. 모수가 쌓이면 잠재고객(Leads) 전환 캠페인 — 맞춤 타겟(데워진 사람)에게 메신저 상담(Lead) 유도
  4. 이긴 것 스케일 + 유사타겟(1~3%) 확장 · 빈도 5+면 소재 교체
  5. 판단 기준: 리드(상담) 1건당 비용(CPA) ≤ 20~40만원이면 흑자 (치료 1건 ₩1,000만 기준)

의료광고 규정 (게재 전 확인)

반드시
  • 효과 보장 · "최고/유일/100%" 단정 금지 · 셀럽 얼굴 + 치료효과 직결 금지 · 환자 동의(비식별)
  • Meta 정책: 개인 건강·신체 특성 단정("당신 아이는 키가 작습니다") 금지 → "우리 아이 키 걱정되시나요?" 보편 톤
  • 각국 현지 의료광고 규제 보수적으로 (특히 태국)
