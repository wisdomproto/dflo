# 05. 키 성장을 방해하는 숨은 요인 8가지

> 출처: 우리 아이 키 성장 바이블 (2장: 키 성장을 방해하는 오해와 진실)

## 핵심 메시지
유전 외에 **8가지 숨은 성장 방해 요인**이 존재한다. 대부분 부모의 관심과 노력으로 관리 가능하다.

---

## 1. 식습관 불균형 🍔

### 위험 체크리스트
- [ ] 탄산음료·과일주스: 하루 1회 이상
- [ ] 초콜릿·사탕·젤리: 거의 매일
- [ ] 달콤한 빵·과자: 주 3회 이상
- [ ] 라면·국수 (밥 대신): 주 3회 이상
- [ ] 패스트푸드 (치킨, 피자, 햄버거): 주 2회 이상
- [ ] 튀김류: 주 2회 이상
- [ ] 가공육 (햄, 소시지, 베이컨): 거의 매일
- [ ] 인스턴트·초가공 식품: 주 3회 이상
- [ ] 아침 식사 거르기: 주 3회 이상
- [ ] 야식 (밤 9~10시 이후): 주 2회 이상

### 개선 원칙
- 저녁 식사는 취침 **최소 3시간 전** 마무리
- 한 입에 **30회 이상** 씹기
- 단 음식은 **주 1~2회**로 제한
- 생선(오메가3): 주 **1~2회**
- 견과류: 매일 한 줌

## 2. 수면 문제 😴

### 성장호르몬 분비의 과학
- 하루 성장호르몬의 **60~70%**가 깊은 수면(비렘 3단계) 중 분비
- **잠든 후 1~2시간**(수면 전반 1/3)에 가장 많이 분비
- 💡 **핵심**: 밤 10시라는 시간보다 **수면의 깊이(질)**가 더 중요

### 연령별 권장 수면
| 연령 | 권장 시간 |
|------|---------|
| 신생아 (0~3개월) | 14~17시간 |
| 영아 (4~11개월) | 12~15시간 |
| 유아 (1~2세) | 11~14시간 |
| 학령 전기 (3~5세) | 10~13시간 |
| 학령기 (6~13세) | **9~11시간** |
| 청소년 (14~17세) | **8~10시간** |

### 수면 방해 요인
- 스마트폰 침실 사용: 매일 평균 **21분 수면 감소** (UC 버클리 연구)
- 월간 ~10시간, 연간 **120시간+** 수면 손실
- 블루라이트 → 멜라토닌 억제 → 수면의 질 저하

### 숙면 환경
- 디지털 커퓨: 취침 **1~2시간 전** 스크린 차단
- 침실 온도: **18~22°C**
- 어둡고 조용한 환경
- 아침 기상 후 **5~10분** 햇빛 쬐기 (체내 시계 리셋)

## 3. 만성 염증성 질환 🤧
- 아토피, 비염, 천식, 장 질환 등
- 염증 사이토카인 → IGF-1 생산 억제 → 성장 저해
- 적절한 질병 + 성장 관리 시: 예상보다 **5~7cm 더** 클 수 있음
- 3~6개월 간격 성장 모니터링 필요

## 4. 잘못된 자세 🪑
- 나쁜 자세: 실제 키보다 **2~3cm** 작아 보임 (심할 경우 3~5cm)
- 장기화 시 근골격계 통증·변형 유발
- 매일 **10~15분** 스트레칭 권장

## 5. 만성 스트레스 😰
- 코르티솔(스트레스 호르몬) → 성장호르몬 분비 직접 억제
- 스트레스 징후: 짜증, 공격성, 틱, 손톱 물기, 야뇨, 식욕 변화, 두통, 복통

## 6. 환경 호르몬 (내분비 교란물질) ⚗️
- 위험 플라스틱: **3번, 6번, 7번**
- 영수증의 비스페놀A (BPA)
- 향료·섬유유연제의 프탈레이트
- 화장품·세제의 파라벤, 트리클로산

## 7. 비만 & 저체중 ⚖️
- **BMI 85+ 백분위**: 과체중, **95+**: 비만 → 전문가 상담
- **BMI 5 미만**: 저체중 → 전문가 상담
- 비만: 뼈나이 가속 → 성장판 조기 폐합
- 지방세포 렙틴 → 조기 사춘기 스위치 ON
- 성조숙증 예방 단백질 상한: 체중 kg당 **최대 1.5g/일**

## 8. 중금속 🔋
- 만 3~5세 남아: 높은 수은 → 성장 지연
- 만 6~11세 남아: 높은 카드뮴 → 비만 위험 ↑
- 납: 성장호르몬 분비와 성호르몬 체계에 영향
- 철분과 칼슘이 납 흡수를 차단
- 해독 식품: 녹차분말, 해조류(알긴산), 마늘, 된장, 도토리묵
