# 09. 잠이 키를 키운다 — 수면과 성장

> 출처: 우리 아이 키 성장 바이블 (5장: 수면 관리)

## 핵심 메시지
성장호르몬의 **60~70%**는 깊은 잠(비렘 3단계) 중에 분비된다. 밤 10시라는 시간보다 **수면의 깊이(질)**가 더 중요하다.

---

## 성장호르몬 분비의 과학
- 하루 분비량의 **60~70%**가 깊은 수면 중 집중 분비
- **잠든 후 1~2시간** (수면 전반 1/3)에 가장 많이 분비
- 밤 10시~새벽 2시 사이가 피크 — 단, **깊이 잠든 상태**여야 함

> 💡 "밤 10시에 잠들어야 한다"보다 **"깊이 잠드는 것"**이 핵심

---

## 연령별 수면 권장 시간

| 연령 | 권장 시간 |
|------|---------|
| 신생아 (0~3개월) | 14~17시간 |
| 영아 (4~11개월) | 12~15시간 |
| 유아 (1~2세) | 11~14시간 |
| 학령 전기 (3~5세) | 10~13시간 |
| **학령기 (6~13세)** | **9~11시간** |
| **청소년 (14~17세)** | **8~10시간** |

---

## 수면 부족의 대가
- 스마트폰 침실 사용: 매일 **평균 21분** 수면 감소 (UC 버클리 연구)
- 월간 약 10시간, 연간 **120시간+** 수면 손실
- 블루라이트 → 멜라토닌 억제 → 깊은 잠 방해
- 수면 부족 → 렙틴(포만감) ↓ + 그렐린(식욕) ↑ → 과식 → 비만 → 조기 사춘기 → 성장판 조기 폐합

---

## 숙면 5원칙

### 1. 일정한 수면 스케줄
- 매일 같은 시간에 취침/기상
- 주말에도 **±1시간 이내** 유지

### 2. 디지털 커퓨
- 취침 **1~2시간 전** 스마트폰·TV·태블릿 차단
- 침실에 전자기기 두지 않기

### 3. 최적의 침실 환경
- 온도: **18~22°C**
- 어둡고 조용한 환경 (암막 커튼 또는 수면 안대)
- 적정 습도: 40~60%

### 4. 아침 햇빛 리셋
- 기상 후 **5~10분** 직사광선 쬐기
- 체내 생체 시계 리셋 → 밤에 더 깊이 잠들게 도움

### 5. 취침 전 루틴
- 잠들기 전 따뜻한 물 목욕
- 카페인 차단: 취침 **3~4시간 전**
- 허기 시: 따뜻한 우유 반 컵 또는 바나나 반 개 (취침 **1.5~2시간 전**)
- 마그네슘 풍부 식품 (견과류, 녹색 채소) 도움

---

## 피해야 할 것
- 취침 전 격렬한 운동
- 침실에서의 공부/놀이 (침실 = 잠자는 곳)
- 늦은 카페인 (초콜릿, 녹차, 콜라 포함)
