# 03. 최종 키 예측 방법

> 출처: 우리 아이 키 성장 바이블 (1장: 우리 아이는 키가 얼마나 클까요?)

## 핵심 메시지
최종 키 예측은 여러 방법을 **종합적으로** 활용해야 하며, 어린아이일수록 오차가 크다. 예측 자체보다 **오늘의 관리**가 더 중요하다.

---

## 방법 1: 표준 성장 도표 추세 분석
> 정확도: ★★★★☆ (높음)

- 6개월~1년 간격으로 키를 측정하고 성장 곡선의 추세를 확인
- 곡선이 특정 백분위를 따라가면 그 백분위의 18세 도달 키를 추정
- **오차**: 어린이 ±8~10cm, 사춘기 후반 **±4~6cm**

### 곡선 해석
| 패턴 | 의미 |
|------|------|
| 50 백분위 유지 | 평균 키로 성장 예상 |
| 곡선이 위로 올라감 | 또래보다 빠르게 성장 중 |
| 곡선이 아래로 꺾임 | 성장 둔화, 전문의 상담 필요 |

---

## 방법 2: 타너 공식 (부모 키 기반, Mid-Parental Height)
> 정확도: ★★★☆☆ (중간)

### 공식
- **아들**: (아빠 키 + 엄마 키 + 13) ÷ 2
- **딸**: (아빠 키 + 엄마 키 - 13) ÷ 2

### 특징
- 집에서 쉽게 계산 가능, 비용 없음
- **오차 범위: ±10cm** (같은 부모 형제자매도 키가 다를 수 있음)
- 환경 요인(영양, 수면, 운동) 미반영
- 조부모 유전 미반영

---

## 방법 3: 뼈나이(골연령) 검사 (X-ray)
> 정확도: ★★★★★ (가장 높음) — 성장 클리닉의 핵심 검사

- 왼쪽 손목 X-ray 촬영 → 뼈 성숙도를 표준 사진과 비교
- 성장판이 얼마나 열려 있는지, 남은 성장 시간 추정
- 사춘기 진행 정도 파악

### 뼈나이 해석
| 결과 | 의미 |
|------|------|
| 뼈나이 = 실제 나이 | 정상 성장 |
| 뼈나이 < 실제 나이 | 늦게 자라는 체질, 성장 여지 있음 |
| 뼈나이 > 실제 나이 | 빨리 자라는 체질, 성장판 빨리 닫힐 수 있음 |

### 예측 정확도 (연령별)
- 어린아이: **±8~10cm** (변수가 많음)
- 사춘기 후반: **±4~6cm** (변수 줄어듦)

---

## 방법 4: 유전자 분석 (최신 기술)
> 정확도: ★★★☆☆ (발전 중)

- 수천 개의 유전자가 복합적으로 키 결정
- 최근 연구: 약 **1만 2천 가지** 유전자 변이가 발견됨 (국내 연구진 김범준 교수팀: 300만개+ 변이 추가 발견)
- 현재까지 밝혀진 변이들로 키 차이의 약 **40%**만 설명 가능
- 유전적 요인의 **48%**는 여전히 미지의 영역
- 현재는 뼈나이 검사의 **보조적 수단**

---

## 예측 정확도의 한계

> "아이의 최종 키를 정확히 '몇 cm다'라고 단정하기는 매우 어렵습니다."

- 사춘기 시작/진행 속도가 최종 키에 매우 큰 영향
- 조기 사춘기: 초반 잘 크다 → 성장판 일찍 닫힘 → 예측보다 작아질 수 있음
- 늦은 사춘기: 일시적으로 작아도 → 더 오래 자람 → 예측보다 클 수 있음

## 조부모 성장 패턴의 중요성
- **체질성 성장 지연**: 늦게 크는 체질, 가족 내 유전적으로 전달
- 할머니 초경 연령이 손녀 성장 예측에 어머니보다 **더 높은 상관관계** (r=0.78)
- 격세유전 가능성 시사
- 진료 시 양가 조부모 성장·사춘기 정보 반드시 확인

## 가장 확실한 전략
> **"오늘의 관리"** — 균형 잡힌 영양, 깊은 잠, 즐거운 운동, 긍정적인 마음
