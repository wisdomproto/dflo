## 팩트체크 기준
- 모든 수치는 책 원문 기반
- 책에 없는 수치는 포함하지 않음 (기존 JSON의 근거 없는 수치 제거)
- 주요 수정 사항:
  - 유전자 수: ~~800개~~ → **1만 2천 가지** 변이
  - 환경 요인 세부 비율(30/25/25/20%): 책에 없으므로 **삭제**
  - 사춘기 시작: 여아 ~~10~12세~~ → **10~11세**, 남아 ~~12~14세~~ → **12~13세**
  - 수면: 시간보다 **깊이(질)**가 더 중요하다는 뉘앙스 보강
  - 관리 효과: ~~+3cm~~ → **5~10cm** (저자의 임상 경험)
