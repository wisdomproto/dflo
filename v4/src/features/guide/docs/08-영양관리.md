# 08. 키 크는 핵심 영양소

> 출처: 우리 아이 키 성장 바이블 (5장: 영양 관리)

## 핵심 메시지
성장에 필수적인 핵심 영양소를 연령에 맞게 **균형 있게** 섭취하는 것이 중요하다.

---

## 핵심 영양소 7가지 (2020 한국인 영양소 섭취기준)

### 1. 단백질 🥩
> 뼈의 50%는 콜라겐(단백질) 골격 + 근육 성장 + 성장호르몬 생성

| 연령 | 1일 권장량 |
|------|---------|
| 6~8세 | **35g** |
| 9~11세 | **50g** |
| 12~14세 (남) | **60g** |
| 12~14세 (여) | **55g** |

- 식품: 살코기, 생선, 달걀, 콩·두부, 우유·치즈, 견과류
- 💡 매 끼니 **손바닥 크기** 단백질 섭취
- ⚠️ 성조숙증 예방: 체중 kg당 **최대 1.5g/일**

### 2. 칼슘 🥛
> 뼈와 치아의 99% 구성 + 골밀도 강화 + 성장판 활성화

| 연령 | 1일 권장량 |
|------|---------|
| 6~8세 | **700mg** |
| 9~11세 | **800mg** |
| 12~14세 | **900mg** |

- 식품: 우유·유제품 (1~2잔/일), 멸치·뱅어포, 케일·브로콜리, 두부
- 💡 우유 2컵 + 멸치 한 줌 ≈ 하루 권장량

### 3. 비타민D ☀️
> 칼슘 흡수 촉진 + 뼈 건강 + 면역력 강화

| 연령 | 1일 권장량 |
|------|---------|
| 전 연령 | **10μg (400IU)** |

- **최고의 방법**: 하루 **15~30분** 직사광선 (팔다리 노출)
- 식품: 연어·고등어, 달걀 노른자, 말린 표고버섯, 강화 우유

### 4. 아연 🥜
> 성장호르몬 활성화 + 세포 분열 촉진

| 연령 | 1일 권장량 |
|------|---------|
| 6~8세 | **7mg** |
| 9~11세 | **9mg** |
| 12~14세 (남) | **10mg** |
| 12~14세 (여) | **9mg** |

- 식품: 굴, 육류, 견과류

### 5. 철분 💉
> 산소 운반 + 성장 에너지 공급

| 연령 | 1일 권장량 |
|------|---------|
| 6~8세 | **9mg** |
| 9~11세 | **11mg** |
| 12~14세 (남) | **14mg** |
| 12~14세 (여) | **16mg** |

- 식품: 적색 육류, 시금치, 두부

### 6. 비타민A 🥕
> 뼈 성장·리모델링 + 면역력

| 연령 | 1일 권장량 |
|------|---------|
| 6~8세 | **450μg RE** |
| 9~11세 | **600μg RE** |
| 12~14세 (남) | **750μg RE** |
| 12~14세 (여) | **650μg RE** |

- 식품: 당근, 고구마, 시금치, 달걀

### 7. 비타민C 🍊
> 콜라겐 합성 + 철분 흡수 촉진 + 면역력

| 연령 | 1일 권장량 |
|------|---------|
| 6~8세 | **60mg** |
| 9~11세 | **80mg** |
| 12~14세 | **100mg** |

- 식품: 딸기, 키위, 감귤, 파프리카, 브로콜리

---

## 수분 섭취 💧
> 신체의 약 70%가 수분

| 연령 | 하루 권장량 |
|------|---------|
| 1~3세 | 500~600ml |
| 4~8세 | 800~1000ml |
| 9~13세 | 1200~1600ml |
| 14~18세 | 1400~1800ml |

---

## 유제품 권장
- 하루 **1~2컵** 우유/유제품
- 새로운 음식 수용: **10~15회** 반복 노출 필요
