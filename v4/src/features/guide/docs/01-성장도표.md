# 01. 성장 도표 읽는 법

> 출처: 우리 아이 키 성장 바이블 (1장: 표준 성장 도표의 놀라운 힘)

## 핵심 메시지
성장 도표는 **비용이 적게 들면서도** 우리 아이의 성장 상태를 **가장 객관적이고 정확하게** 파악할 수 있는 강력한 도구이다.

## 백분위수 이해하기
- **백분위수 50**: 100명 중 50번째, 평균적인 성장
- **백분위수 10**: 100명 중 10번째로 작은 편
- **백분위수 3 미만**: 저신장 위험 신호, 전문의 상담 필요

> 💡 **핵심**: 백분위수 숫자 자체보다 **곡선의 흐름(추세)**이 더 중요하다!

## 정상 성장 신호
- 성장 곡선이 특정 백분위선을 따라 **꾸준히 상승**
- 6개월~1년 간격 측정 시 곡선이 일정하게 유지
- 백분위 3~97 사이를 안정적으로 유지

## 주의가 필요한 신호 (적신호)
- 성장 곡선이 **갑자기 아래로 꺾임** (예: 50→25→10)
- 백분위가 2단계 이상 떨어짐
- 백분위 3 미만으로 지속
- **만 3세 이후 연간 4cm 미만** 성장

## 성장 곡선의 진단적 가치
| 곡선 패턴 | 의심 가능한 원인 |
|-----------|-----------------|
| 꾸준히 작지만 곡선을 잘 따라감 | 가족성 저신장 또는 체질성 성장 지연 |
| 정상 크다가 갑자기 성장 멈춤 | 성장호르몬 결핍, 갑상샘 기능 저하증 |
| 키에 비해 체중 과다 | 갑상샘 문제, 쿠싱 증후군 감별 필요 |

## 성장 도표 활용법
1. **정기적 측정**: 6개월~1년 간격으로 키와 몸무게 측정
2. **정확한 기록**: 같은 시간대, 같은 조건에서 (아침, 공복)
3. **점 찍기**: 성장 도표에 나이와 키를 표시
4. **선 연결**: 여러 점을 이어 성장 곡선 완성
5. **추세 확인**: 곡선의 방향과 기울기 관찰
