1. 프로젝트 개요

🎯 프로젝트 목표

연세새봄의원 187 성장 클리닉의 여러 아이를 동시에 관리할 수 있는 모바일 최적화 성장 관리 웹 애플리케이션

✨ 핵심 가치
  • 여러 아이의 성장 기록을 한 앱에서 체계적으로 관리
  • 아이별 맞춤형 성장 곡선 자동 표시
  • 과학적 근거 기반 예상키 측정 (Khamis-Roche 방법)
  • 날짜별 챌린지 관리 및 성취도 시각화
  • 오프라인 사용 가능 (PWA)

👥 타겟 사용자

  • 주 사용자: 2-18세 자녀를 둔 부모님
  • 특히: 여러 명의 자녀를 관리하는 부모님
  • 환경: 모바일 기기 (스마트폰, 태블릿)
  • 사용 빈도: 주 1-2회 (기록), 매일 (챌린지)

3. 여러 아이 관리 시스템 NEW v2.0

🎯 핵심 컨셉

한 가정의 여러 자녀를 하나의 앱에서 독립적으로 관리할 수 있는 시스템

주요 기능

  • 아이 추가/편집/삭제: 이름, 성별, 생년월일 관리
  • 아이 선택: 카드 터치로 빠른 전환
  • 데이터 분리: 성장 기록, 챌린지 데이터 완전 독립
  • 선택 표시: 선택된 아이는 파란색 테두리 강조
  • 자동 전환: 모든 페이지가 선택된 아이 데이터로 자동 업데이트
💡 사용자 경험
부모가 형제자매의 데이터를 혼동하지 않고, 각 아이의 성장을 체계적으로 추적할 수 있습니다.

📱 UI/UX

홈 화면 - 아이 선택 카드

  • 아이별 카드: 아바타(아이콘), 이름, 나이, 성별 표시
  • 선택된 아이: 파란색 테두리 + Teal 그라데이션 배경
  • 편집 버튼(✏️): 정보 수정
  • 삭제 버튼(🗑️): 아이 정보 및 모든 데이터 삭제
  • 추가 버튼(➕): 새 아이 추가

  • 성장진단: "👦 김민수의 성장 기록"
  • 챌린지: "👧 김지우의 챌린지"
  • 현재 선택된 아이를 명확하게 표시

4. 성장 진단

📝 성장 기록 입력

자동화된 입력 시스템 ENHANCED v2.0

  • 날짜: 사용자 입력 (오늘 날짜 기본값)
  • 키 (cm): 사용자 입력
  • 몸무게 (kg): 사용자 입력
  • 성별: 선택된 아이 정보로 자동 설정 ✨
  • 만 나이: 생년월일과 날짜로 자동 계산 ✨
✨ 날짜별 자동 나이 계산
날짜 입력을 변경하면 해당 날짜 기준 만 나이가 실시간으로 자동 계산됩니다. 과거 기록 추가 시에도 정확한 나이가 자동으로 반영됩니다.

화면 표시

  • 입력 필드: 날짜, 키, 몸무게 (한 줄 배치)
  • 정보 배지: 👦 남아 / 👧 여아, 만 X세 (한 줄 배치)
  • 성별별 색상: 남아(파란색), 여아(분홍색)
  • 나이 배지: Teal 그라데이션

📊 성장 곡선 차트

KCDC 2017 표준 성장곡선

  • 백분위선: P5, P10, P25, P50, P75, P90, P95 (7개 전체 표시)
  • 연령 범위: 2-18세
  • 데이터 출처: 질병관리청 2017 소아청소년 성장도표

자동 성별 그래프 NEW v2.0

  • 선택된 아이의 성별에 맞는 그래프만 자동 표시
  • 남아 선택 시: 파란색 남아 그래프
  • 여아 선택 시: 분홍색 여아 그래프
  • 성별 탭 자동 숨김 (수동 전환 불필요)

차트 기능

  • 키/몸무게 탭 전환
  • 사용자 데이터 점으로 표시 (8px, 흰색 테두리)
  • 백분위 자동 계산 및 라벨 표시
  • 모바일 최적화 (400px 높이)

📏 예상키 측정 (Khamis-Roche 방법)

적용 대상

  • 4-17세 아동
  • 예측 범위: ±5cm (개인차 고려)

필요 입력 데이터

  • 아이: 성별, 나이, 현재 키, 현재 몸무게
  • 부모: 아버지 키, 어머니 키

계산 원리

  1. 중간 부모 키 (MPH) 계산
    • 남아: (아버지 키 + 어머니 키 + 13) ÷ 2
    • 여아: (아버지 키 + 어머니 키 - 13) ÷ 2
  2. 현재 성장 상태 평가
    • KCDC 2017 표준 데이터로 현재 백분위 계산
    • 백분위 높음 → 유전적 예상키보다 클 가능성
  3. 나이별 가중치 적용
    • 어릴수록 부모 키 영향 큼
    • 나이가 들수록 현재 키 영향 큼

결과 표시

  • 예상 최종 키 (대형 숫자)
  • 예상 범위 (±5cm)
  • 계산 과정 투명하게 표시
  • 주의사항 안내
💡 예상키 탭 최적화
예상키 측정 탭 선택 시 성장 곡선 차트와 기록 목록이 자동으로 숨겨져 예상키 계산에만 집중할 수 있습니다.
