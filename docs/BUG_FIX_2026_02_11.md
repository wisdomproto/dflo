## 🎯 수정 완료 이슈 (3개)

### 1️⃣ manifest.json start_url 이슈 수정 ✅

**문제**:
```
manifest.json:1 Manifest: property 'start_url' ignored, should be same origin as document
```

**원인**: 
- `start_url`이 절대 경로(`/index.html`)로 설정되어 있어 브라우저에서 경고 발생

**해결**:
```json
// 수정 전
"start_url": "/index.html",

// 수정 후
"start_url": "./index.html",
```

**영향**: 
- PWA 설치 시 시작 URL이 올바르게 인식됨
- 브라우저 콘솔 경고 제거

**문제**:
```
korea-growth-standard.js: 잘못된 유형: weight | 가능한 값: height, weight
```

**영향**:

- 성장 분석 기능 정상 작동

### 3️⃣ Apple 메타 태그 deprecated 경고 수정 ✅

**문제**:
```
<meta name="apple-mobile-web-app-capable"> is deprecated
```

**원인**: 
- iOS 13+ 버전부터 자동으로 PWA 지원
- 해당 메타 태그가 더 이상 필요하지 않음

**해결**:
- 모든 HTML 파일에서 deprecated 메타 태그 제거
- 설명 주석으로 대체

**수정된 파일**:
- `home.html`
- `routine.html`
- `cases.html`
- `growth.html`

**변경 내용**:
```html
<!-- 수정 전 -->
<meta name="apple-mobile-web-app-capable" content="yes">

<!-- 수정 후 -->
<!-- iOS PWA 지원 (iOS 13+에서 자동 지원됨) -->
```
