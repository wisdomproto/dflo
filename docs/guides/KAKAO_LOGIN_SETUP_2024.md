# 🟡 카카오 로그인 설정 가이드 (2024 최신 버전)

## 📌 단계별 설정 방법

### 1단계: 카카오 로그인 활성화

1. **Kakao Developers 콘솔 접속**
   - https://developers.kakao.com/
   - 앱 선택 (LPCare)

2. **제품 설정 → 카카오 로그인**
   - 왼쪽 메뉴에서 "제품 설정" 클릭
   - "카카오 로그인" 선택
   - **"활성화 설정" ON**

---

### 2단계: Redirect URI 설정

**같은 페이지에서 아래로 스크롤**

1. **Redirect URI 섹션**
   - "Redirect URI 등록" 버튼 클릭
   
2. **개발용 URI 추가**
   ```
   http://localhost:8000
   http://localhost:8000/login.html
   http://127.0.0.1:8000
   ```

3. **배포용 URI 추가** (나중에)
   ```
   https://your-domain.com
   https://your-domain.com/login.html
   ```

---

### 3단계: Web 플랫폼 설정

**두 가지 방법 중 하나:**

#### 방법 A: 카카오 로그인 페이지에서
1. 같은 "카카오 로그인" 페이지에서
2. **"Web 플랫폼 등록"** 섹션 찾기
3. 사이트 도메인 입력:
   ```
   http://localhost:8000
   ```

#### 방법 B: 앱 설정에서
1. 왼쪽 메뉴 → **"앱 설정"**
2. **"플랫폼"** 또는 **"Web"** 클릭
3. "Web 플랫폼 등록" 버튼 클릭
4. 사이트 도메인 입력:
   ```
   http://localhost:8000
   ```

---

### 4단계: 동의 항목 설정

1. **제품 설정 → 카카오 로그인 → 동의 항목**
2. 아래 항목 설정:

| 항목 | 설정 | 용도 |
|------|------|------|
| 닉네임 | 필수 동의 | 사용자 이름 |
| 프로필 사진 | 선택 동의 | 프로필 이미지 |
| 카카오계정(이메일) | 선택 동의 | 이메일 주소 |

---

### 5단계: JavaScript 키 확인

1. **앱 설정 → 앱 키**
2. **JavaScript 키** 복사
3. `config.js`에 입력:

```javascript
const CONFIG = {
    KAKAO_JS_KEY: '여기에-복사한-키-붙여넣기',
    // ...
};
```

## 🚨 문제 해결

### "플랫폼" 메뉴가 안 보여요
**해결:**
- 최신 UI에서는 "카카오 로그인" 페이지에 통합됨
- "제품 설정 → 카카오 로그인"에서 모든 설정 가능

### Redirect URI 오류
```
RedirectUriMismatch error
```
**해결:**
1. 카카오 콘솔 Redirect URI 확인
2. 정확히 `http://localhost:8000` 입력
3. `https` vs `http` 구분
4. 포트 번호 확인 (8000)

### JavaScript 키를 찾을 수 없어요
**해결:**
1. 앱 설정 → 앱 키
2. "JavaScript 키" 복사
3. ⚠️ REST API 키가 아님!
