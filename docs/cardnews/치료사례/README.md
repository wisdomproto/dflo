# 치료사례 카드뉴스 (홈페이지 공개 4개)

홈페이지 치료사례 캐러셀에 노출 중인 **공개 4개 케이스**(`cases.html?only=3,4,6,7`)를
인스타그램 카드뉴스로 재구성한 **초안 스펙**입니다. 시스템의
`marketing_cardnews_slides` 스키마(`docs/cardnews/data/*.json` 와 동일 포맷)를 그대로 따릅니다.

- `case-3-성조숙증비만.json` — 👦 성조숙증·비만 (초진 예상 162.9 → 최종 185.8cm, 현재 177.4cm)
- `case-4-부모키작음.json` — 📏 부모 키가 작은 경우 (171.2 → 182.2cm, 아빠 키 돌파)
- `case-6-여아성조숙.json` — 👧 여아·성조숙증, 미국 남매 (153.8 → 165.3cm)
- `case-7-성장느림.json` — ✈️ 성장이 느린 아이, 일본 방학 (167 → 183.6cm, 현재 175.6cm, 유전키 돌파)

소스: `v4/i18n/locales/ko.yml` 의 `realcases.cards` (홈페이지 공개 스탯) + 클리닉 진료 프레이밍.
**모든 예상키·태그·한 줄 설명은 이미 홈페이지에 공개된 문구를 그대로 사용**(새 임상 수치 창작 없음).

## 각 파일 구조 (data/*.json 과 동일)
```jsonc
{
  "sortOrder": null,          // 62 토픽 밖 커스텀 콘텐츠 → 번호 미배정 (임포트 시 지정)
  "title": "…",
  "category": "치료사례",
  "caption_ko": "…", "caption_en": "", "caption_th": "", "caption_vi": "", "caption_ch": "",
  "hashtags_ko": "#…", "hashtags_en": "", …,
  "slides": [
    { "n": 1, "role": "표지", "illustration": "[레이아웃]…[색]…",
      "headline_ko": "…", "headline_en": "", …,
      "subtext_ko": "…",  "subtext_en": "", …,
      "isCTA": false }
  ]
}
```

## 언어
지금은 **한국어(ko)만** 채워져 있습니다. en/th/vi/ch/cn 은 빈 문자열(`""`)로 자리만 마련.
다국어 확장 시 스튜디오 AI 번역 또는 별도 transcreation 패스로 채우면 됩니다.
번역 화자 = **남성 한국인 의사** → 태국어 `ครับ/ผม` 등 격식·남성 종결 규칙 준수
(루트 CLAUDE.md `feedback_i18n_speaker_register`).

## 공용 일러스트 스타일 (모든 슬라이드 공통 — 프롬프트엔 장면 고유 요소만)
- 플랫 2D 벡터, 세로 2:3 (1024×1536)
- 보라 그라데이션 `#667eea → #764ba2` 배경, 강조색 민트 `#33D6B5`
- 같은 7~9세 한국 아이 캐릭터 유지
- **case-6 (여아)**: 캐릭터=여아, 강조색 핑크 `#ec4d8b` (홈페이지 케이스 6 테마색과 일치)
- CTA 슬라이드 일러스트엔 **로고·도메인·"로고 자리" 언급 금지** — 로고/도메인은 디자이너가 별도 오버레이

## 유튜브 치료후기 영상 연결 (⚠️ 확인 필요)
각 공개 케이스에는 R2 `website.json` 의 `youtubeUrl`(치료후기 영상)이 붙어 있습니다.
이 실행 환경에서 **youtube.com 이 네트워크 정책으로 차단**되어 나레이션을 자동으로 읽지 못했습니다.
환자명↔영상 추정 매핑(`cases/update_youtube_urls.mjs` 기준):

| 케이스 | 고민 | 추정 환자 | 유튜브 영상 | 비고 |
|--------|------|-----------|-------------|------|
| case 3 | 성조숙증·비만 | (확인 필요) | — | 매핑 미확정 |
| case 4 | 부모 키 작음 | 재윤(이재윤) | 없음 | 원본에 영상 링크 없음 |
| case 6 | 여아·성조숙 남매 | 세희 | `youtu.be/MSxSk8NePVc` | 오빠(지훈)와 같은 영상 |
| case 7 | 성장 느림·일본 | 고키(공개명 제임스) | `youtu.be/K-IVd1k-7sg` | 윤우와 같은 영상 |

→ case 6·7 파일에는 **"유튜브 후기" 슬롯 슬라이드**를 넣어 두었습니다(현재는 영상 안내 문구).
영상 자막/나레이션을 붙여주시면 **부모 인용 한 줄**을 그 슬라이드에 반영해
신뢰도(사회적 증거)를 높일 수 있습니다.

## 의료광고법 / 톤 준수
- 정보성 톤. "187이 치료합니다" 식 강한 홍보·효과 단정·공포 조장 금지.
- 수치는 **예상키(projected)** 로만 표기 + 개인차 disclaimer(캡션 말미 `※`).
- "주사" 단어 금지 → "사춘기 관리/속도 조절" 로 서술(PATIENT_STORY_GUIDE 규칙).

## 임포트 (선택)
`docs/cardnews/import-to-db.mjs` 는 62 토픽(`sort_order` 매칭)을 전제로 하므로 그대로는 안 됩니다.
치료사례 카드뉴스는 **커스텀 콘텐츠**라 임포트하려면 먼저 `marketing_articles`(kind='custom') 행을
만들어 `content_id` 를 연결해야 합니다 — 이 초안 단계에서는 스펙 파일만 제공합니다.
