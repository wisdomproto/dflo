> **형식**: 정보성 카드뉴스 7장 · 플랫 벡터 · GPT-4o 이미지 생성용 (텍스트 포함)

## 사용법
1. 아래 **[공통 스타일]** + **[7장 일러스트]** 가 이미지 골격입니다 (모든 언어 공통).
2. 만들 언어의 **[텍스트]** 섹션에서 그 장의 「헤드라인」·「보조문구」·「도메인」을 가져와, 일러스트 프롬프트의 `[헤드라인]`/`[보조문구]`/`[도메인]` 자리에 넣어 GPT-4o에 1장씩 생성.
3. ① 생성 후 **그 이미지를 레퍼런스로 첨부** → "이 아이·스타일 그대로 ②:…" 로 이어가면 7장이 한 세트로 유지됨.
4. 로고는 ⑦번 상단 중앙에 **직접 첨부**.
5. ⚠️ GPT가 그린 한글/숫자/도메인 **오타·깨짐 검수 필수**. 틀리면 *"텍스트만 「정확한문구」로 고쳐줘"*.

## 공통 스타일 (모든 카드 동일 적용)
```
· 비율: 세로 2:3 (1024×1536)
· 아트 스타일: 플랫 2D 벡터 일러스트. 굵고 둥근 형태, 단색 면(그라데이션 없음)에 옅은 드롭섀도우만. 사진·3D 아님.
· 배경: 부드러운 보라색 세로 그라데이션 (#667eea 위 → #764ba2 아래), 넓은 여백.
· 캐릭터: 모든 장에 같은 7~9세 한국 어린이(밝은 표정, 파스텔 옷). 장마다 동일 인물 유지.
· 색: 텍스트는 흰색, 포인트 색은 민트(#33D6B5) 절제 사용.
· 텍스트: 굵은 둥근 고딕(Noto Sans 느낌), 또렷하게, 잘림·오타 없이. 지정된 문구만 넣고 다른 글자 추가 금지.
· 피할 것: 사실적 사진, 무섭거나 우울한 분위기, 복잡한 배경, 잘린 글자, 깨진 글자, 워터마크, (7번 외) 임의 로고.
```

## 7장 일러스트 프롬프트 (언어 공통 — 텍스트만 언어별로 교체)

### ① 표지
```
[레이아웃] 상단 30%는 헤드라인 영역(비움), 중앙~하단에 인물.
[주체] 키 큰 부모와 옆에 선 아이. 아이는 손바닥을 자기 머리 위에 얹어 부모 키와 비교하며 살짝 갸웃하는 표정.
[디테일] 둘의 키 차이가 또렷이 보이게. 아이 머리 위 작은 물음표 1개. 바닥에 옅은 그림자.
[배경] 인물 뒤에 아주 옅은 점선 키 눈금 1줄.
[색] 인물 파스텔, 물음표만 민트.
[헤드라인]  ← 언어별 텍스트 ①
```

### ② 80:20의 진짜 의미
```
[레이아웃] 상단 헤드라인 / 중앙 큰 숫자 대비 / 하단 보조문구.
[주체] 화면 중앙에 '80'과 '20'을 큰 숫자로. 80은 차분한 회보라로 다소 흐리게, 20은 민트색으로 또렷하고 위에 작은 스포트라이트 빛.
[디테일] 20 옆에서 아이가 손가락으로 20을 가리키는 포즈.
[배경] 장식 최소.
[색] 20=민트 강조, 80=회보라.
[헤드라인]  ← 언어별 텍스트 ②
[보조문구]  ← 언어별 텍스트 ②
```

### ③ 유전 = 잠재력
```
[레이아웃] 중앙~상단에 큰 산, 하단 왼쪽에 작게 선 아이.
[주체] 아이가 산 아래에서 한 손으로 햇빛 가리며 정상을 올려다봄, 기대에 찬 표정, 3/4 뒷모습.
[디테일] 둥근 보라 계열 산봉우리, 정상에 민트 깃발. 아래에서 정상으로 위를 향한 점선 화살표. 옅은 구름 2개.
[배경] 단순한 하늘.
[색] 깃발·화살표 민트.
[헤드라인]  ← 언어별 텍스트 ③
[보조문구]  ← 언어별 텍스트 ③
```

### ④ 환경 = 실현하는 힘
```
[레이아웃] 산 중턱을 오르는 구도, 헤드라인 상단.
[주체] ③과 같은 아이가 산길을 씩씩하게 오름. 작은 배낭(영양·휴식 상징), 한 발 내딛는 동작.
[디테일] 산길에 작은 계단/이정표, 위쪽에 민트 깃발이 보임. 아이 주변에 살짝 반짝이는 효과.
[배경] ③과 같은 산(연속성 유지).
[색] 깃발·반짝임 민트.
[헤드라인]  ← 언어별 텍스트 ④
[보조문구]  ← 언어별 텍스트 ④
```

### ⑤ 임상 사례  (※ 키 수치 명시)
```
[레이아웃] 화면을 좌우 둘로 나누고 가운데 얇은 구분선.
[주체] 왼쪽: 높은 목표(점선)지만 실제는 낮게 멈춘 시무룩한 아이 / 오른쪽: 낮은 출발이지만 더 높이 오른 밝은 아이.
[디테일] 각 쪽에 작은 성장 곡선 — 왼쪽 곡선은 회색, 끝점에 "150cm" 라벨 / 오른쪽 곡선은 민트색, 끝점에 "170cm" 라벨. 목표선은 점선.
[배경] 좌우 톤을 아주 살짝 다르게.
[색] 오른쪽 민트 강조.
[헤드라인]  ← 언어별 텍스트 ⑤
[보조문구]  ← 언어별 텍스트 ⑤
```
> 숫자 **150 / 170 (cm)** 는 모든 언어 공통으로 이미지에 그대로 표기.

### ⑥ 환경 관리 4가지
```
[레이아웃] 상단 헤드라인 / 중앙 2×2 격자 둥근 카드 4개 / 하단 보조문구.
[주체] 둥근 카드 4개: (1) 균형 식사 접시 (2) 초승달+잠든 얼굴 (3) 줄넘기하는 아이 (4) 편안한 미소/하트. 각 카드 아래 작은 라벨.
[디테일] 카드는 흰 반투명, 아이콘은 파스텔+민트 포인트, 간격 균일.
[배경] 단순.
[색] 아이콘 포인트 민트.
[헤드라인]  ← 언어별 텍스트 ⑥
[보조문구]  ← 언어별 텍스트 ⑥ (영양·수면·운동·스트레스 4개 라벨)
```

### ⑦ 결론  (※ 로고 + 도메인)
```
[레이아웃] 상단 중앙 로고 / 그 아래 헤드라인 / 중앙 인물 / 하단 도메인.
[로고] 상단 중앙에 첨부한 로고 이미지를 배치할 것.
[주체] 환하게 웃으며 한 뼘 더 자란 아이와 흐뭇하게 바라보는 부모. 아이는 두 팔 벌려 시원하게 키 큰 포즈.
[디테일] 아이 옆에 위로 향한 새싹/화살표(성장 상징). 따뜻하고 희망적인 분위기, 옅은 반짝임.
[배경] 밝고 환한 톤.
[색] 포인트 민트.
[헤드라인]  ← 언어별 텍스트 ⑦
[보조문구]  ← 언어별 텍스트 ⑦
[도메인]    ← 언어별 도메인 (하단, 작은 흰색 텍스트)
```
