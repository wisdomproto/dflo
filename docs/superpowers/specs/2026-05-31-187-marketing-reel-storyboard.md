## 4. 씬별 상세

### S1 · 키 성장·골든타임 인트로 🆕
- **비주얼**: **작은 아이 vs 큰 아이 대비**(감정 후크) — 유지. 비주얼은 대비로 시선 끌고, 카피만 보편화.
- **카피**: **"우리 아이 키 성장, 지금이 바로 골든타임입니다"**
- **타겟 노트**: "또래보다 작은"은 작은 아이 부모만 → 타겟 좁음. 대부분 아이는 평균이고 **평균에 만족하는 부모는 없음** → "우리 아이 키 성장" 보편 메시지로 전체 부모 타겟.
- **자산**: AI 이미지 1~2장 (§7). 신규.

### S2 · 원장·실적·병원 통합 신뢰 ♻️🔧 (구 S2+S4 합침 = 원래대로)
- **비주얼**: 기존 ClinicScene — 좌상 원장 초상 + 우상 실적 카운트업(1,000+ / 95%+) + 하단 병원 사진 슬라이딩 + Korea 배지. 여기에 **해외 학회(말레이시아·태국)** 추가.
- **카피(강)**: **"대한민국이 주목하는 강남 187 성장클리닉"** / "채용현 원장 · 국제 학회 초청 강연" / "성장 진료 10년 · 1,000+ 케이스" / 도달률 **95%+**
- **자산**: `director.png` + 병원 사진 재활용 + 학회 사진(없으면 배지). 신규 `PositioningScene` 폐기(여기로 흡수).
