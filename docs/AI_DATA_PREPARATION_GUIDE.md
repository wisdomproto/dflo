## 📁 데이터 파일 구조

```
data/
├── training/
│   ├── faq.json              # 질문-답변 1000개+
│   ├── knowledge_base.json   # 의학 지식
│   ├── cases.json            # 치료 사례 100개+
│   ├── guidelines.json       # 성장 가이드
│   └── protocols.json        # 진료 프로토콜
│
├── reference/
│   ├── growth_charts.json    # KCDC 성장 곡선 데이터
│   ├── percentiles.json      # 백분위수 테이블
│   └── normal_ranges.json    # 정상 범위
│
└── context/
    ├── clinic_info.json      # 클리닉 정보
    └── policies.json         # 상담 정책
```

## 💡 데이터 품질 체크리스트

### 필수 확인 사항:
- [ ] 의학적으로 정확한 정보
- [ ] 전문의 검토 완료
- [ ] 최신 가이드라인 반영
- [ ] 개인정보 제거 (환자 이름 등)
- [ ] 일관된 톤앤매너
- [ ] 병원 정보 포함
- [ ] 법적 책임 제한 문구

### 데이터 다양성:
- [ ] 다양한 나이대 (3세~18세)
- [ ] 남아/여아 균형
- [ ] 다양한 증상/질문 유형
- [ ] 긍정적/부정적 사례
- [ ] 일상적/긴급 상황

### 절대 하지 말아야 할 것:
