# 🌱 187 성장케어 플랫폼 v3.0

> 연세새봄의원 187 성장 클리닉의 **모바일 최적화 아이 성장 관리 웹 애플리케이션** (완전 재설계 버전)

---

## 🎯 **프로젝트 개요**

### **v3.0 주요 변경사항**

| 항목 | v2.0 (기존) | v3.0 (신규) |
|------|-------------|-------------|
| **아키텍처** | 22개 파일, 중복 코드 70% | MVC + Service 패턴, 11개 핵심 모듈 |
| **챌린지** | 운동 체크리스트 | ✅ **데일리 루틴** - 식사/수면/운동/영양제 통합 |
| **데이터 입력** | 수동 1명씩 | ✅ **스프레드시트 대량 입력** |
| **초진 정보** | 기본 정보만 | ✅ **상세 설문지** (남아/여아) |
| **UI/UX** | PC/모바일 혼재 | ✅ **Mobile-First** (관리자만 PC) |
| **사진 업로드** | 없음 | ✅ **식사 사진** (Supabase Storage) |
| **운동 가이드** | 텍스트 | ✅ **유튜브 영상 연동** |


## 🏗️ **아키텍처**

### **MVC + Service Pattern**

```
📁 src/
├── core/                    # 공통 모듈 (utils, constants, api)
├── services/                # 비즈니스 로직
├── components/              # 재사용 UI 컴포넌트
├── controllers/             # 페이지 컨트롤러
└── pages/                   # HTML 페이지

기술 스택:
- Frontend: Vanilla JS (ES6 모듈), HTML5, CSS3
- Backend: Supabase (PostgreSQL, Auth, Storage)
- 배포: Cloudflare Pages
- 차트: Chart.js
```

## 📱 **주요 화면**

### **1. 데일리 루틴** ★ 신규 핵심 페이지

```
routine.html
├── 신체 측정 (키/몸무게)
├── 식사 기록 (사진 + 텍스트) - 아침/점심/저녁/간식
├── 수면 (취침/기상/수면질)
├── 수분 섭취 (ml, 진행바)
├── 영양제 (기본/추가)
├── 성장 주사
├── 운동 (유튜브 영상 연동)
└── 메모 & 기분

뷰 전환:
- 입력 모드
- 캘린더 뷰 (월간 완료 현황)
- 통계 뷰 (주간/월간 분석)
```

### **2. 설문지** ★ 신규

```
questionnaire.html
├── 8개 섹션 (진행률 표시)
│   ├── 기본 정보
│   ├── 과거 성장 기록 (8세~현재)
│   ├── 건강 상태
│   ├── 2차 성징 (남아/여아 별도)
│   ├── 생활 습관
│   ├── 식습관
│   ├── 운동 습관
│   └── 가족력
└── 임시 저장 & 수정 가능
```

### **3. 관리자 - 스프레드시트 대량 입력** ★ 신규

```
tools/bulk-import.html
├── 엑셀 데이터 복사 & 붙여넣기
├── 자동 파싱 & 검증
├── 미리보기
└── 대량 업로드 (진행 상황 실시간)
```
