## 📱 **새로운 데일리 루틴 페이지**

### **주요 기능**

#### **1. 3가지 뷰 모드**
```
[입력 모드]  [달력]  [통계]
    ●           ○      ○
```

- **입력 모드**: 오늘의 루틴 기록
- **달력 뷰**: 월간 완료 현황
- **통계 뷰**: 주간/월간 분석

---

#### **2. 입력 섹션 (9개)**

| 섹션 | 아이콘 | 내용 |
|------|--------|------|
| 신체 측정 | 📏 | 키(선택) / 몸무게(필수) |
| 식사 기록 | 🍽️ | 아침/점심/저녁/간식 (사진+텍스트+건강도) |
| 수면 | 😴 | 취침/기상 시간, 수면 질 |
| 수분 섭취 | 💧 | 물 섭취량 (ml), 진행바 |
| 영양제 | 💊 | 비타민D/칼슘/아연/종합 + 추가 |
| 성장 주사 | 💉 | 맞았는지 체크 + 시간 |
| 운동 | 🏃 | 운동 이름 + 시간 (분) |
| 메모 | 📝 | 오늘 하루 메모 |
| 기분 | 😊 | 좋음/보통/나쁨 |

---

#### **3. 데이터 저장**

- **LocalStorage 사용**
- **저장 키**: `routine_YYYY-MM-DD`
- **데이터 형식**: JSON

```javascript
{
    date: "2026-02-05",
    height: 145.2,
    weight: 38.5,
    sleepTime: "22:00",
    wakeTime: "07:00",
    sleepQuality: "good",
    waterAmount: 1500,
    supplements: ["vitamin-d", "calcium"],
    injection: true,
    injectionTime: "21:00",
    meals: {
        breakfast: { ... },
        lunch: { ... },
        dinner: { ... },
        snack: { ... }
    },
    exercises: [ ... ],
    notes: "오늘 하루 메모",
    mood: "good"
}
```

## 🎨 **UI/UX 개선**

### **챌린지 vs 데일리 루틴 비교**

| 항목 | 챌린지 (기존) | 데일리 루틴 (신규) |
|------|--------------|-------------------|
| **범위** | 운동만 | 신체/식사/수면/영양제/운동 통합 |
| **데이터** | Boolean | 상세 데이터 + 메모 |
| **입력** | 체크박스 | 다양한 입력 형식 |
| **시각화** | Streak | 달력 + 통계 + 진행바 |
| **분석** | 없음 | 주간/월간 통계 |

---

### **새로운 UI 컴포넌트**

1. **날짜 선택기**
   - 이전/다음 날짜 버튼
   - 날짜 피커

2. **뷰 모드 토글**
   - 입력/달력/통계 전환
   - Active 상태 표시

3. **식사 카드**
   - 4개 식사 (아침/점심/저녁/간식)
   - 완료 상태 표시
   - 모달로 상세 입력

4. **수분 트래커**
   - 진행바
   - +250ml / +500ml 버튼
   - 목표 2000ml

5. **품질 선택기**
   - 수면 질: 좋음/보통/나쁨
   - 기분: 😊/😐/😞

## 🚀 **테스트 방법**

### **1. 로컬 서버 실행**
```bash
python -m http.server 8000
```

### **2. 로그인**
```
URL: http://localhost:8000/login.html
부모 ID: 1
비밀번호: 1234
```

### **3. 데일리 루틴 접근**
- 홈 화면 하단 네비게이션
- "📝 데일리루틴" 탭 클릭

### **4. 기능 테스트**

✅ **날짜 선택**
- ◀/▶ 버튼으로 날짜 변경
- 날짜 클릭해서 피커 사용

✅ **신체 측정**
- 몸무게 입력 (필수)
- 키 입력 (선택)

✅ **식사 기록**
- 4개 식사 카드 클릭
- 모달에서 시간/내용/건강도/식사량 입력
- 저장 후 "✓ 완료" 표시 확인

✅ **수분 섭취**
- +250ml / +500ml 버튼
- 진행바 업데이트 확인
- 초기화 버튼

✅ **영양제**
- 체크박스 선택
- 추가 영양제 입력

✅ **운동**
- "+ 운동 추가하기" 클릭
- 이름/시간 입력
- 목록에 표시 확인
- 삭제 버튼

✅ **저장**
- "💾 저장하기" 버튼 클릭
- 알림 확인
- 다른 날짜로 이동 후 돌아와서 데이터 유지 확인
