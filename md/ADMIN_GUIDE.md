# 관리자 페이지 사용 가이드

## 🎯 접근 방법

1. **홈 화면**에서 환영 배너의 **"⚙️ 관리자"** 버튼 클릭
2. **ID/PW 입력 불필요** - 바로 관리자 페이지로 이동

---

## 💬 FAQ 관리

### FAQ 추가하기
1. **"💬 FAQ 관리"** 탭 선택
2. **"➕ FAQ 추가"** 버튼 클릭
3. 입력 항목:
   - **카테고리**: 성장, 운동, 영양, 수면, 자세, 기타 중 선택
   - **질문**: 사용자가 자주 묻는 질문 (예: "키가 크려면 얼마나 자야 하나요?")
   - **답변**: 질문에 대한 답변 내용
4. **"저장"** 클릭

### FAQ 수정/삭제
- **수정**: FAQ 항목의 **"✏️ 수정"** 버튼 클릭 → 내용 수정 → 저장
- **삭제**: FAQ 항목의 **"🗑️ 삭제"** 버튼 클릭 → 확인

---

## 📊 치료 사례 관리

### 사례 추가하기
1. **"📊 치료 사례"** 탭 선택
2. **"➕ 사례 추가"** 버튼 클릭
3. **환자 정보 입력**:
   - 이름: 김민수
   - 성별: 남아/여아 선택
   - 생년월일: 2015-03-15

4. **측정 기록 추가**:
   - 기본 1개 측정 입력란 제공
   - **"➕ 측정 추가"** 버튼으로 추가 측정 입력
   - 각 측정마다:
     - 날짜: 2024-01-15
     - 키: 135.5 cm
     - 몸무게: 32.0 kg
   - 만 나이는 생년월일과 측정 날짜로 **자동 계산**

5. **메모 입력** (선택):
   ```
   - 6개월 성장호르몬 치료
   - 꾸준한 운동과 영양 관리
   - 총 9cm 성장 (135cm → 144cm)
   ```

6. **"저장"** 클릭

### 사례 상세 보기
1. 사례 카드 클릭
2. 표시 내용:
   - **환자 정보**: 이름, 성별, 생년월일, 만 나이
   - **성장 곡선 차트**:
     - 파란색 곡선: KCDC 2017 표준 (P5, P50, P95)
     - 빨간색 점: 환자의 측정 데이터
   - **측정 기록 테이블**: 날짜별 키/몸무게
   - **메모**: 치료 과정 및 특이사항

3. **"수정"** 버튼: 사례 정보 수정
4. **"닫기"** 버튼: 모달 닫기

### 사례 수정/삭제
- **수정**: 사례의 **"✏️ 수정"** 버튼 또는 상세보기 → **"수정"** 버튼
- **삭제**: 사례의 **"🗑️ 삭제"** 버튼 클릭 → 확인

## 📈 성장 곡선 차트 해석

### 차트 구성
- **X축**: 만 나이 (2세 ~ 18세)
- **Y축**: 키 (cm)
- **파란색 곡선 3개**:
  - **P5** (점선): 하위 5% - 작은 편
  - **P50** (실선, 굵게): 중간값 - 평균
  - **P95** (점선): 상위 5% - 큰 편
- **빨간색 점 + 선**: 환자의 실제 측정 데이터

### 판단 기준
- **P50 근처**: 평균적인 성장
- **P5 이하**: 저신장 의심, 전문 상담 권장
- **P95 이상**: 큰 편, 조기 성장 여부 확인
- **곡선 따라가기**: 일정한 백분위를 유지하면 정상 성장
- **곡선 이탈**: 갑자기 위/아래로 벗어나면 주의 필요

## 💾 데이터 저장

### 저장 위치
- **브라우저 로컬스토리지** (localStorage)
- **키**:
  - `adminFaqs`: FAQ 목록
  - `adminCases`: 치료 사례 목록

### 데이터 백업
```javascript
// 콘솔에서 실행
// 백업
const backup = {
  faqs: localStorage.getItem('adminFaqs'),
  cases: localStorage.getItem('adminCases')
};
console.log(JSON.stringify(backup));
// 복사해서 저장

// 복원
localStorage.setItem('adminFaqs', backup.faqs);
localStorage.setItem('adminCases', backup.cases);
location.reload();
```

## 🚀 빠른 시작

### 1단계: FAQ 3개 추가
- 성장 관련 FAQ 1개
- 운동 관련 FAQ 1개
- 영양 관련 FAQ 1개

### 2단계: 치료 사례 2개 추가
- 남아 사례 1개 (3회 이상 측정)
- 여아 사례 1개 (3회 이상 측정)

### 3단계: 확인
- 정보 페이지에서 FAQ 표시 확인
- 치료 사례 페이지에서 사례 표시 확인
- 관리자 페이지에서 차트 정상 작동 확인

## 🔧 문제 해결

### FAQ가 정보 페이지에 안 보여요
- `localStorage`에 `adminFaqs` 키가 있는지 확인
- 콘솔: `JSON.parse(localStorage.getItem('adminFaqs'))`

### 치료 사례가 사례 페이지에 안 보여요
- `localStorage`에 `adminCases` 키가 있는지 확인
- 콘솔: `JSON.parse(localStorage.getItem('adminCases'))`

### 차트가 안 그려져요
- Chart.js가 로드되었는지 확인
- 측정 기록이 1개 이상 있는지 확인
- 생년월일과 측정 날짜가 올바른지 확인

### 데이터를 초기화하고 싶어요
```javascript
// 콘솔에서 실행
localStorage.removeItem('adminFaqs');
localStorage.removeItem('adminCases');
location.reload();
```
