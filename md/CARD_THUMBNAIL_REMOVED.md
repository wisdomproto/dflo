## 🎯 변경 사항

### Before (이전):
- ❌ 카드에 썸네일 이미지 표시
- ❌ 모달에도 이미지 표시

### After (수정):
- ✅ **카드: 아이콘만 표시** (썸네일 없음)

## 📸 이미지 표시 위치

### 1️⃣ **카드 (목록)** - **이미지 없음** ❌
```
┌─────────────────┐
│   📊 아이콘     │
│   제목          │
│   부제목        │
│   ⏱️ 5분       │
│   #태그1 #태그2 │
└─────────────────┘
```

### 2️⃣ **모달 (상세)** - **이미지 있음** ✅
```
┌─────────────────────┐
│   📊 아이콘         │
│   제목              │
│   부제목            │
├─────────────────────┤
│   [큰 이미지]       │  ← 여기만 표시!
├─────────────────────┤
│   읽기 시간/카테고리│
│   요약              │
│   핵심 포인트       │
│   ...               │
└─────────────────────┘
```
